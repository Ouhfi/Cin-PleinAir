# 🎬 Movie App (Netflix Inspired)

## 📌 Description
This project is a Netflix-inspired movie website built using HTML, CSS, and JavaScript.  
It allows users to explore movies, view details, and enjoy a clean and modern interface.

## 🚀 Features
- 🎥 Browse movies from an API (TVMaze)
- ⭐ View movie titles and ratings
- 🎲 Random movie selection
- 💻 Responsive and modern UI
- ⚡ Fast and simple user experience

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- TVMaze API
